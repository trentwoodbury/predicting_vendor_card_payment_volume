training_split_percent = 0.8
test_split_percent = 1 - training_split_percent

dataset.set_index('organization_id', inplace=True)

training_columns = [col for col in dataset.columns if col not in eval_columns]

X = dataset.loc[:, training_columns]
month_1 = dataset.loc[:, 'month_1_cpv']
month_2 = dataset.loc[:, 'month_2_cpv']
month_3 = dataset.loc[:, 'month_3_cpv']

row_shuffle = np.random.choice(range(X.shape[0]), replace=False, size=X.shape[0])
cutoff_index = int(training_split_percent * row_shuffle.shape[0])

training_row_indices = row_shuffle[:cutoff_index]
test_row_indices = row_shuffle[cutoff_index:]

X_train = X.iloc[training_row_indices, :].values

m2_train = month_2.iloc[training_row_indices].values

m1_test = month_1.iloc[test_row_indices].values

Removing Outliers¶

Our big MAE may be coming from outliers, so let's try removing those from the training dataset. I don't know where to set the outlier threshold, so let's just try different thresholds and see which one has the lowest MAE.

In [17]:
thresholds = np.arange(90,100,0.5)
maes = []

for threshold in thresholds:
    X_train_3 = X.iloc[training_row_indices, :]
    X_test_3 = X.iloc[test_row_indices, :].values
    outlier_threshold = np.percentile(X_train_3, threshold)
    inclusion_filter = [True if val < outlier_threshold else False for val in X_train_3['cpv'].values]

    X_train_3 = X_train_3.loc[inclusion_filter, :].values
    y_train_3 = m1_train[inclusion_filter]

    gb = XGBRegressor(objective="reg:squarederror", n_estimators=2)
    gb.fit(X_train_3, y_train_3)

    predictions = gb.predict(X_test_3)
    errors = [np.abs(actual - pred) for pred, actual in zip(m1_test, predictions)]
    maes.append(np.average(errors))
    
plt.plot(thresholds, maes)
plt.title("MAE as a function of outlier cutoff")
plt.ylabel("Mean Absolute Error")
plt.xlabel("Outlier Cutoff Percentile")
plt.show()

It looks like the top performance is when we don't remove any outliers. I'm assuming the 95 plummet is a fluke from removing a single datapoint since it is such an outlier from the overall trend line. I would verify this given more time.

Month 1 Model Conclusions¶

For month 1, our best model is XGBoost setting n_estimators to 2 with no outlier removal.
